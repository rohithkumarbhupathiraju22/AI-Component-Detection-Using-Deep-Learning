# 🔍 AI-Based Electronic Component Detection

An intelligent deep learning system that detects and classifies electronic components from images with high accuracy.

📌 Overview

This project uses a **Convolutional Neural Network (CNN)** to identify various electronic components from images.
It is designed to assist beginners, students, and engineers in quickly recognizing components using AI.

🚀 Features

* ✅ Multi-class component detection
* ✅ High accuracy (~98%)
* ✅ Fast prediction
* ✅ Easy-to-use notebook interface
* ✅ Works on custom images

 🧠 Model Details

* **Model Type:** CNN (Deep Learning)
* **Framework:** TensorFlow / Keras
* **Training Platform:** Google Colab
* **Accuracy Achieved:** ~98%

🧩 Classes Detected

* IC
* Arduino
* Capacitor
* Jumper Wire
* LED
* Push Button
* Resistor
* Ultrasonic Sensor

🛠️ Tech Stack

* Python
* TensorFlow / Keras
* OpenCV
* NumPy
* Google Colab
